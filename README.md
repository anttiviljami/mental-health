# mental-health

Small reminders for your mental health ♥

```
npm i mental-health
```

## Usage

```
$ npx mental-health
♥: Did you sleep well?
```

## Installation

To get a reminder each time you open a new terminal window, run it in your shell init script e.g.

```
echo -e '\n# small mental health reminder ♥\nnpx mental-health' >> ~/.profile
```
